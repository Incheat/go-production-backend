# Repository Visibility Guide

## Options

* **Public** — visible to everyone.
* **Private** — visible only to you or selected collaborators.

## Choosing Public

Use when:

* The project is educational.
* You want community contributions.
* Licensing is clear.

## Choosing Private

Use when:

* Project contains sensitive info.
* It's experimental or incomplete.
* It belongs to a closed-source product.

## Tips

* You can change visibility later.
* Always remove secrets before publishing.
